# 🎧 Spotify User Listening Behavior Analysis
*What does listening behavior reveal about user engagement?*

Using SQL, this project explores Spotify streaming history to uncover listening patterns, playback behavior, and engagement insights that can support product decisions.


## 💼 Business Context

Every interaction on Spotify - playing a song, skipping a track, enabling shuffle, or switching devices, creates valuable behavioral data.

For product teams, these interactions provide insight into how users engage with the platform, helping improve recommendations, personalize user experiences, and identify opportunities to increase engagement.

This project analyzes Spotify streaming history using SQL to explore the patterns hidden within everyday listening behavior.
